## Setup

1. Download all files and maintain the folder structure above
2. Add your own images to the `images/` folders using the exact filenames listed
3. Upload everything to your GitHub repository
4. Enable GitHub Pages (Settings → Pages → Deploy from main branch)

## Images Needed

## GitHub Pages Deployment

1. Push all files to your repository
2. Go to Repository Settings → Pages
3. Under "Source" select your main branch, root folder
4. Your site will be live at: https://[yourusername].github.io/[reponame]/
